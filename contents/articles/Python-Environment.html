venv

[TOP]

venv is a built-in module in python3.3 or higher.

Pros: Build-in module no need package installation.

Cons: Cannot create environment in different python version.

Commonly used commands:

python -m venv <env_name>
・To create new virtual environment.

<env_name>\Scripts\activate.bat
・To Active virtual environment.
